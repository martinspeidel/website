  • Add Colors

    Add Colors adds the Color Fill Effect to multiple selected layers.

    • Save your color palettes as global presets
    • Different color fill modes such as Repeat, Random, Step, or Blend
    • Add keyframes to the Color Fill Effect using shortcuts
    • Responsive interface



    Download


  • Add Copies

    Add Copies adds copies of selected layers.

    • Duplicate any number of selected layers with customizable offsets
    • Apply offsets to multiple properties such as Position, Scale, Rotation, Opacity, and Path
    • Create a controller null with sliders for all selected properties
    • Generate copies along a customizable path
    • Responsive Interface



    Download


  • Add Delay

    Add Delay adds animation delay to copies of layers.

    • Select a main layer and the copies or layers you want to affect
    • Choose the properties you want to affect and press the + button
    • Set the delay amount in the effects panel. Voila!






    Download


  • Add Folders

    Add Folders adds a folder structure to your After Effects project.

    • Add your personal folder structure to your project
    • Define your own folder templates in a XML-File and share it with your team






    Download


  • Add Grid

    Add Grid adds pixel perfect guides and grids to your compositions.

    • Save and name your guides as global presets
    • Divide your grid in X and/or Y independently
    • Add guides to an objects anchorpoint
    • Set guides position by %
    • Change the color of the guides
    • *.jsxbin shortcuts included (works with KBar)






    Download


  • Add Parent Null

    Add Parent Null is a script to attach and parent selected layers to a null object.

    • Automatic layer length and timeline position of null object
    • Automatic color label grouping
    • Center anchor point on selected layers
    • Change shape, color, thickness, size, position of the null object
    • The null object's own anchor point is centered
    • No more useless null solids in the project manager
